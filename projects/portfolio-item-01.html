  • HTML5
  • CSS3
  • DOM Manipulation

~Demo // Source Code~

Simple weather app that makes calls OpenWeather Api using fetch.

Customizable search, so that user can choose their units of measurement, city and the duration that they will be out of the house.

Displays the city and country with the current temperature and a short description of the current weather conditions. Will also conditionally display a message to the user which informs them whether they will need an umbrella based on the time they have indicated.
